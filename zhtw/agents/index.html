資料庫
對應職位：資料分析師 / DBA

想看數字不用再排隊等工程師。用白話文在你的通訊軟體裡問「上個月哪個產品賣最好」，它自己摸資料庫、找對的表、跑出結果 — 你不需要知道 schema 長什麼樣子。

  • 自然語言查詢，自動探索 schema
  • 支援多種資料庫（PostgreSQL / MySQL ...）
  • 跨資料庫聯合查詢
  • 結構化結果輸出（JSON / CSV）
