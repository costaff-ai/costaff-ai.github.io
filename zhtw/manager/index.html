CoStaff Manager

只要開口，
它便統籌整個團隊。

CoStaff Manager 是 CoStaff 的核心協調者，也是你直接對話的助理。它理解你的需求，將工作分派給合適的專家 Agent，安排提醒與例行任務，再將完成的成果送回你的對話中。

01  它做什麼

一段對話，背後是一整個團隊。

你只需與單一助理對話，Manager 便會在背後調度整個團隊。

與單一助理對話

無論使用哪種聊天軟體都能連上；在同一處，什麼都能交辦。

分派工作給專家

它理解你的需求，將工作交給合適的 Agent，例如程式開發、商業分析，或你自行加入的 Agent。

Hub-and-spoke 協調

Manager → Agent A → Manager → Agent B → Manager → 你。專家在背後彼此協作，你只會看到整理好的成果。

提醒與例行任務

「下午三點提醒我」或「每天早上彙整 AI 新聞」，它會排定時間，再將結果推送至你的聊天室。

02  建立在開放標準上

CoStaff 的開源核心。

不受單一供應商綁定：Manager 為開源，並採用整個生態共通的協定。

Google ADK

以 Google ADK 的 LlmAgent 打造，採純協調者模式，核心接線約 30 行。

A2A 協定

透過開放的 A2A 協定與專家 Agent 溝通，你也能接上自訂的 Agent。

MCP 工具

以 MCP server 處理排程、通知與檔案等作業。

自建且開源

於你自有的機器上執行，即為 CoStaff 的核心，並以 AGPL v3 開源。

03  開始使用

部署你專屬的 Manager。

三行指令，即可從零建立可運作的 Manager 與團隊，全程執行於你自有的硬體。

$curl -fsSL https://raw.githubusercontent.com/costaff-ai/costaff/main/install.sh | bash
