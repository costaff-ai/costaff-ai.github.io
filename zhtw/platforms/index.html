CoStaff Platforms

一個帳號，11 套企業系統，
AI Agent 都會操作

自架的企業套件，與 CoStaff Agent 生態深度整合 — 每套平台都內建 MCP server，你的 AI 員工能直接替你在 ERP 過傳票、查 CRM 客戶、開客服工單。全套為私有原始碼，聯絡取得。

全套件共通的治理能力

單一登入 SSO

Keycloak 級的自建身分中心 — OIDC、PKCE、MFA、Passkey。登入一次，全套系統通行。

Per-app 角色

同一個人在 ERP 是業務、在 CRM 唯讀 — 全部集中在一個地方指派。

三 Portal 介面

員工、主管、管理三種視角，每套平台依照權限自動呈現對的畫面。

可編輯權限矩陣

角色 × 模組的讀寫權限，直接在介面上調整、即時生效 — 不必重新部署。

通知中心 + 個人 API Token

簽核與指派即時通知；個人 token 只帶本人權限，不多不少。

Agent-operable

每套平台一個 MCP server 加一個配對 Agent，AI 員工用跟你一樣的方式操作系統。

Account Manager
定位：身分中心 / SSO 樞紐

Keycloak 風格的身分提供者，全套系統登入一次搞定。一個帳號、一張權限矩陣、所有應用 — 稽核需要的紀錄也都齊全。

  • OIDC + PKCE 授權流程
  • MFA、Passkey、Google 登入
  • users × apps 應用權限矩陣
  • SSO 工作階段管理、稽核匯出、Webhooks

ERP
定位：企業營運核心

鼎新風格的 ERP，11 個模組把採購到付款、接單到收款一條龍走完 — 從報價到總帳、跨多公司別，關鍵單據都有簽核鏈把關。

  • 採購 / 銷售 / 庫存 / BOM-MRP
  • AR-AP、總帳、電子發票、多公司別
  • 報價轉單 + 信用額度管控
  • 委外加工、品檢、簽核鏈、自動化規則

從第一次進站到簽約：潛客評分、商機看板拖拉推進、加權銷售預測 — 客戶完整歷史就在一條時間軸上。

  • 潛客評分 + 商機看板
  • 報價 / 訂單 / 合約、加權銷售預測
  • 客戶 360 時間軸 + Email 範本

出勤、薪資、績效一站搞定 — 內建法規申報試算（勞退、二代健保、扣繳憑單），月底不用再回 Excel 算一遍。

SCM

把供應商拉進同一套流程：RFQ 詢比議價到三方比對、需求預測自動補貨，供應商入口網讓對帳變成協同作業。

  • RFQ 詢比議價 + 三方比對
