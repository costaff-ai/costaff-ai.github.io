CoStaff 文件

在你自己的機器上自架多代理 AI 系統。文件帶你跑完安裝、設定、加入專家 Agent 與通訊軟體。

CoStaff 是多代理協作系統：Manager Agent 透過 A2A 派工給專家 Agent（商業分析、程式開發、資料庫、自訂等），專家再用 MCP 呼叫工具完成工作。整套跑在你機器上的 Docker。

想趕快動手？直接到 安裝。想先看大圖？技術架構頁說明六層架構。

四行快速上手

已經熟 stack，就是這四行：

# 1. 安裝
curl -fsSL https://raw.githubusercontent.com/costaff-ai/costaff/main/install.sh | bash

# 2. 設定（互動式精靈 — 沒有手動步驟時安裝程式會自動接著跑）
source ~/.zshrc  # Ubuntu 改 ~/.bashrc
costaff onboard

# 3. 啟動全部（會先 preflight 檢查 .env）
costaff start

# 4. 開 dashboard，點 Chat 開聊
costaff dashboard

接著加 Agent 與 channel — 看 專家 Agent 與 通訊軟體。
