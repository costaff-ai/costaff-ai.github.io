01 — 入門

系統需求

安裝 CoStaff 前主機上需要準備的東西。

項目最低建議
作業系統macOS 12+ / Ubuntu 20.04+macOS 14+ / Ubuntu 22.04+
CPU2 vCPU（x86_64 或 ARM64）4 vCPU
記憶體8 GB16 GB
硬碟20 GB 可用50 GB（保留給 Agent 工作目錄與 Docker image）
Python3.12（安裝程式會處理）—
DockerEngine 24+ / Desktop 4.30+—
LLMGoogle API key、OpenAI 相容 endpoint，或本機 Ollama—

主機上會被裝進去的東西

  • Python 3.12 — macOS 用 Homebrew，Ubuntu 用 deadsnakes PPA。只有 costaff CLI 自己會用。
  • Docker — macOS Docker Desktop、Ubuntu Docker Engine。所有 Agent 與 channel 都跑在容器裡。
  • 虛擬環境 — ~/.costaff/.venv/，裡面 pip install -e 裝好 CLI。

檔案在哪

  • ~/.costaff/costaff/ — 主 repo 的 git clone（CLI、Manager Agent、MCP core）。
  • ~/.costaff/costaff-agent/<name>/ — 每個 Agent 的原始碼與 .env。
  • ~/.costaff/costaff-channel/<name>/ — 每個 channel 的原始碼與 .env。
  • ~/.costaff/workspace/ — bind mount 到容器的 /app/data。Agent 讀寫使用者檔案都在這。
完全本機跑

CoStaff 不會偷偷打 API 回家。對外的網路請求只有你設定的 LLM 服務 — 而且只要把 LLM 換成 Ollama 或其他本機 endpoint，連這通都免了。
