02   關於

住在你機器上的 AI 員工

自架部署、記憶你的工作習慣、透過你已有的通訊 App 與你對話。

01
記得你

姓名、工作、習慣、偏好——不用每次重新說明。長期記憶讓每次對話都更有效率。

02
住在你的 App 裡

Telegram、Discord、LINE、WebChat——它出現在你已經在用的地方，不需要養成新習慣。

03
跑在你的機器

自架於 Mac Mini 或任何 Linux 主機。資料從不外流，你完全掌控。開源、透明。

03   如何開始

三步上手，第一天就能用

01
安裝 CLI

一行指令完成安裝。只需要 Python 和 Docker——這兩樣大多數開發者都已經有了。

pip install costaff
02
回答幾個問題

互動式 Onboard 精靈帶你選好 AI 模型、設定通訊管道。不用手改設定檔。

costaff onboard
03
開始對話

一個指令啟動所有服務。打開 Telegram，發一句話，你的 AI 員工就在線了。

costaff start
